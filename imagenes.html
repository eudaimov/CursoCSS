Imágenes

Introducción

En esta sección vamos a ver todos los atributos que podemos emplear con los elementos de imágen (<img >)

Atribuos

border-radius

Para bordes redondeados.

<img class="bordesRedondos" src="./img/brujula.webp" alt="Imagen bordes redondeados ejemplo"> .bordesRedondos{ display: block; border-radius: 50%; width: 200px; aspect-ratio: 1/1; margin-left: auto; margin-right: auto; }

border

Para crear un borde, sobre todo en imágenes en miniatura.

<img class="bordes" src="./img/brujula.webp" alt="Imagen bordes redondeados ejemplo" > .bordes{ display: block; margin-right: auto; margin-left: auto; aspect-ratio: 1/1; width: 200px; border-radius: 50%; border: 2px solid blueviolet; padding: 0.5rem; } .bordes:hover{ border: 2px solid orangered; }

box-shadow

Para realizar una sombra alrededor de una imágen.

max-width

Para cambiar el tamaño dinámicamente dependiendo el tamaño de la ventana.

margin-let:auto; margin-right:auto

Para centrar una imagen.

opacity

Para modificar la opacidad de la imagen.

filter

Para colocar un filtro a nuestra imagen.

transform: scaleX(-1)

Para voltear una imagen.

box-reflect: below, above, lefto right. (-webkit-box-reflect:)

Para reflejar una imagen.

object-fit

Es yba propiedad CSS se usa para especificar cómo se debe cambiar el tamaño de un <img> o <video> para que se ajuste a su contenedor.

Si diseñamos la imagen para que tenga la mitad de su ancho y la misma altura, la imagen se verá deformada y aplastada por los lados. Para evitar que suceda esto debemos aplicar la propiedad object-fit.

Las diferentes opciones son:

  • fill. Es la opción por defecto, la imagen se deformará para adaptase a las dimensiones del elemento contenedor.
  • contain: La imagen reducirá su tamaño, pero conservará su relación entre el ancho y alto para adaptarse al tamaño del contenedor
  • cover. La imagen se recortará o ampliará para cubrir las dimensiones del contenedor sin modificar en ningún caso la relación entre el ancho y el alto.
  • none: La imagen se recortará a las dimensiones del contenedor no modificando su tamaño. Si la imagen tiene mayor tamaño que el contenedor tendrá el mismo efecto que cover.
  • scale-down. La imagen se reduce a la versión más pequeña de "none" o "contain"

object-position

La propiedad CSS se usa para especificar cómo se debe colocar un <img> o un <video> dentro de su contenedor.

Se usa junto con object-fit para especificar cómo se debe colocar un elemento (img, video) con las coordenadas x/y dentro de su "propio cuadro de contenido".

offset

La propiedad se usa cuando se anima un elemento a lo largo de una ruta y es una propiedad abreviada para las siguientes propiedades:

  • offset-anchor*(Sólo Firefox)
  • offset-distance
  • offset-path
  • offset-rotate

outline

Borde sin afectar al ancho y alto del elemento

mask-image

Crea una capa de máscara para colocar sobre un elemento para ocultar parcial o totalmente partes del elemento.
