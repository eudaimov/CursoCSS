Display Grid

Introducción

CSS Grid Layout (también conocido como "Grid" o "CSS Grid"), es un sistema de diseño bidimensional basado en cuadrículas que, en comparación con cualquier sistema de diseño web del pasado, cambia por completo la forma en que diseñamos las interfaces de usuario. CSS siempre se ha utilizado para diseñar nuestras páginas web, pero nunca ha hecho un buen trabajo. Primero, usamos tablas, luego flotadores, posicionamiento y bloques en línea, pero todos estos métodos eran esencialmente trucos y dejaban fuera muchas funciones importantes (centrado vertical, por ejemplo). Flexbox también es una excelente herramienta de diseño, pero su flujo unidireccional tiene diferentes casos de uso, y en realidad funcionan juntos.

Grid es el primer módulo CSS creado específicamente para resolver los problemas de diseño que todos hemos estado solucionando desde que creamos sitios web.

Conceptos básicos

  • Contenedor de cuadrícula
  • Filas y columnas
  • Celda
  • Tamaño de las columnas y las filas.
  • Tamaño o espacio de celda
  • Identificación de las celdas
  • Area de celdas
  • Espacio entre celdas
  • Alineación del contenedor
  • Alineación de las celdas

Para comenzar, debe definir un elemento contenedor como una cuadrícula con display: grid, establecer los tamaños de columna y fila con grid-template-columns y grid-template-rows, y luego colocar sus elementos secundarios en la cuadrícula con grid-column y grid-row.

De manera similar a flexbox, el orden de origen de los elementos de la cuadrícula no importa. Su CSS puede colocarlos en cualquier orden, lo que hace que sea muy fácil reorganizar su cuadrícula con consultas de medios. Imagine definir el diseño de toda su página y luego reorganizarla por completo para adaptarse a un ancho de pantalla diferente, todo con solo un par de líneas de CSS. Grid es uno de los módulos CSS más poderosos jamás presentados.

Propiedades del contenedor

display

Define el elemento como un contenedor de cuadrícula y establece un nuevo contexto de formato de cuadrícula para su contenido.

display:grid/ inline-grid

  • grid: Genera una cuadrícula a nivel de bloque
  • grid-inline: Genera una cuadrícula en una línea

grid-template-rows

Define las filas existentes en el grid con una lista de valores separados por espacio.

Los valores pueden representar los nombres de las filas o el tamaño de las filas.

Valores:

  • nombre de línea:

    Van entre corchetes y puede tener un nombre compuesto separado por guiones o incluso varios nombres separado por espacio.

    Los nombres pueden ser identificadores no únicos. Y no son obligatorios, si no asignamos ningún nombre se puede identificar las líneas de columnas por los números que se asignaran automáticamente empezando por el 1 o negativos empezando por la cantidad negativa de columnas siendo la última -1.

    Si varias líneas comparten el mismo nombre, se puede hacer referencia a ellas por su nombre de línea y número.

  • tamaño de fila:

    Puede ser una longitud, un porcentaje o una fracción del espacio libre en la cuadrícula usando la fr unidad

    Es obligatorio.

    La fr unidad le permite establecer el tamaño de una pista como una fracción del espacio libre del contenedor de cuadrícula. Por ejemplo, esto establecerá cada elemento en un tercio del ancho del contenedor de la cuadrícula:

    grid-template-rows:1fr 1fr 1fr

    El espacio libre se calcula después de cualquier elemento no flexible.

    Puede ser una longitud, un porcentaje o una fracción del espacio libre en la cuadrícula usando la fr unidad

grid-template-areas

Define una plantilla de cuadrícula haciendo referencia a los nombres de las áreas de cuadrícula que se especifican con la grid-area propiedad. Repetir el nombre de un área de cuadrícula hace que el contenido abarque esas celdas. Un punto significa una celda vacía. La sintaxis en sí proporciona una visualización de la estructura de la cuadrícula.

Valores:

  • el nombre de un área de cuadrícula especificada con grid-area
  • . – un punto significa una celda de cuadrícula vacía
  • none – no se definen áreas de cuadrícula

Ejemplo:

.container { width: 50vw; margin: 0 auto; display: grid; grid-template-columns: 1fr 1fr 1fr 1fr; grid-template-rows: auto; grid-template-areas: "header header header header" "main main . sidebar" "footer footer footer footer"; text-align: center; } .item-1 { grid-area: header; background-color: orange; } .item-2 { grid-area: main; background-color: dodgerblue; } .item-3 { grid-area: sidebar; background-color: orangered; } .item-4 { background-color: limegreen; grid-area: footer; } .item{ border: 1px solid gray; border-radius: 5px; padding-block: 1rem; }

Eso creará una cuadrícula de cuatro columnas de ancho por tres filas de alto. Toda la fila superior estará compuesta por el área del encabezado . La fila del medio estará compuesta por dos áreas principales , una celda vacía y un área de barra lateral . La última fila es todo pie de página .

Cada fila de su declaración debe tener el mismo número de celdas.

Puede usar cualquier número de puntos adyacentes para declarar una sola celda vacía. Siempre que los puntos no tengan espacios entre ellos, representan una sola celda.

Tenga en cuenta que no está nombrando líneas con esta sintaxis, solo áreas. Cuando usa esta sintaxis, las líneas en cada extremo de las áreas en realidad se nombran automáticamente. Si el nombre de su área de cuadrícula es foo , el nombre de la línea de fila inicial y la línea de columna inicial del área será foo -start , y el nombre de su última línea de fila y última línea de columna será foo -end . Esto significa que algunas líneas pueden tener varios nombres, como la línea del extremo izquierdo en el ejemplo anterior, que tendrá tres nombres: inicio de encabezado, inicio principal e inicio de pie de página.

grid-template

Es una abreviatura para establecer grid-template-rows, grid-template-columns y grid-template-areas en una sola declaración.

Valores:

  • none – establece las tres propiedades en sus valores iniciales(rows, columns and area).
  • [grid-template-rows] / [grid-template-columns] – establece grid-template-columns y grid-template-rows a los valores especificados, respectivamente, y establece grid-template-areas entre comillas en la zona del grid-template-rows

Ver como funciona cada una de las propiedades en particular: grig-template-columns, grid-template-rows y grid-template-areas

Por ejemplo:

grid-template: [row1-start] "header header header" 25px [row1-end] [row2-start] "footer footer footer" 25px [row2-end] / auto 100px auto;

grid-column-gap & grid-row-gap

Especifica el tamaño de las líneas de la cuadrícula. Puede pensar en ello como establecer el ancho de los canalones entre las columnas/filas.

Los canalones solo se crean entre las columnas/filas, no en los bordes exteriores.

La propiedad puede grip-column-gap o column-gap(denominación más moderna), grip-row-gap o row-gap(denominación más moderna). Las propiedades sin prefijo ya son compatibles con Chrome 68+, Safari 11.2 Release 50+ y Opera 54+.

Por ejemplo:

#ejemploGap{ margin: 0 auto ; width: 50vh; display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: auto; column-gap: 1rem; row-gap: 1rem; outline: 5px solid dodgerblue; } .itemGap{ background-color: darkorange; border-radius:5px; border: 2px solid gray; padding: 1rem; }

grid-gap o gap.

Es la abreviatura de row-gap y column-gap.

Se escribe un sólo valor tanto para el columna como para el gap o dos valores.

Si se escriben dos valores, primero se escribe el valor de la fila(row) seguido y separado por un espacio del valor de la columna (column)

gap: 10px 20px

justify-items

Alinea los elementos de la cuadrícula a lo largo del eje en línea (fila) (a diferencia de align-items lo que se alinea a lo largo del eje del bloque (columna) ). Este valor se aplica a todos los elementos de la cuadrícula dentro del contenedor.

Se suele utilizar cuando las unidades utilizadas en grid-template-columns son fr sino se utiliza justify-content

Este comportamiento también se puede establecer en elementos de cuadrícula individuales a través de la justify-self propiedad.

Valores:

  • stretch – llena toda el ancho de la celda (este es el valor predeterminado)
  • start – alinea los elementos para que queden al ras con el borde inicial de su celda
  • end – alinea los elementos para que queden al ras con el borde final de su celda
  • center – alinea los elementos en el centro de su celda

Ejemplo:

#ejemploJustifyItems{ width: 50vw; height: 300px; margin: 0 auto; display: grid; grid-template-columns: 1fr 1fr 1fr ; grid-template-rows: repeat(3, 1fr); outline: 4px dodgerblue solid; justify-items: center; row-gap: 10px; } .itemJustify{ background-color: darkorange; border-radius: 5px; border: 2px solid black; padding-inline: 1rem; }

align-items

Alinea los elementos de la cuadrícula a lo largo del eje vertical (columna/bloque) (a diferencia de justify-items lo que se alinea a lo largo del eje horizontal (fila/línea). Este valor se aplica a todos los elementos de la cuadrícula dentro del contenedor.

Se suele utilizar cuando las unidades utilizadas en grid-template-rows son fr sino se utiliza align-content

Este comportamiento también se puede establecer en elementos de cuadrícula individuales a través de la align-self propiedad.

Ejemplo:

#ejemploAlignItems{ margin: 0 auto; width: 50vh; height: 300px; display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: 1fr 1fr 1fr; outline: 3px dodgerblue solid; align-items: start; column-gap: 10px; } .itemAlignItems{ background-color: darkorange; padding: 1rem; border: 2px solid gray; }

place-items

Es una propiedad abreviada. Establece las propiedades align-items y justify-items en una sola declaración.

El primer valor establece align-items, el segundo valor justify-items. Si se omite el segundo valor, el primer valor se asigna a ambas propiedades.

justify-content

Esto podría suceder si todos los elementos de la cuadrícula tienen el tamaño de unidades no flexibles como px. En este caso, puede establecer la alineación de la cuadrícula dentro del contenedor de la cuadrícula. Esta propiedad alinea la cuadrícula a lo largo del eje en línea (fila)

grid-auto-flow

Si tiene elementos de cuadrícula que no coloca explícitamente en la cuadrícula, el algoritmo de colocación automática se activa para colocar automáticamente los elementos. Esta propiedad controla cómo funciona el algoritmo de colocación automática.

Los valores pueden ser: column, row y dense(realmente no comprendo bien su funcionamiento)

La forma colocación de los elementos por defecto es row, es decir, primero empieza a colocar por las las filas, de izquierda a derecha, pasando posteriormente a la siguiente fila. El modo column empieza a colocar de arriba a abajo empezando por la izquierda.
