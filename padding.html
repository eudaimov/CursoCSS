Márgenes internos o Relleno : Padding

¿Qué es?

El relleno se utiliza para crear espacio alrededor del contenido de un elemento, dentro de los bordes definidos.

Hay propiedades para establecer el relleno para cada lado de un elemento (superior, derecho, inferior e izquierdo), para la zona vertical y para la zona horizontal, o uno global

Ancho y Alto: 200px Padding: 100px

Propiedad abreviada o general

4 valores: superior, derecho, inferior, izquierdo (movimiento de las agujas del reloj)

3 valores: superior, derecho-izquierdo, inferior

2 valores: superior-inferior, derecho-izquierdo

1 valor: cuadro lados.

Inline & block

padding-inline: Se puede colocar dos valores y es el margen interno lateral de una linea(horizontal) influye la dirección (writing-mode & direction)

padding-block: Se pueden colocar dos valores y es el margen vertical de un bloque (vertical) influye la dirección (writing-mode $ direction)-
