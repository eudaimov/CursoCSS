¿Qué es?

Este término se utiliza para hablar de diseño o maquetación en css. Se utiliza para hablar de una caja que se rodea a cualquier elemento que se encuentre dentro del body

Explicación de las diferentes partes:

  • Contenido : el contenido del cuadro, donde aparecen el texto y las imágenes.
  • Relleno : Es un área alrededor del contenido. El relleno es transparente. Dentro del relleno se encuentra el padding y el borde interno o Outline.
  • Borde : un borde que rodea el relleno y el contenido.
  • Margen : borra un área fuera del borde. El margen es transparente.

El modelo de caja nos permite agregar un borde alrededor de los elementos y definir el espacio entre los elementos.

Ancho y alto de un elemento

Para establecer correctamente el ancho y el alto de un elemento en todos los navegadores, debe saber cómo funciona el modelo de caja.

Importante: cuando establece las propiedades de ancho y alto de un elemento con CSS, simplemente establece el ancho y el alto del área de contenido . Para calcular el tamaño completo de un elemento, también debe agregar relleno, bordes.

El ancho total de un elemento debe calcularse así:

Ancho total del elemento = ancho + padding izquierdo + padding derecho + borde izquierdo + borde derecho

La altura total de un elemento debe calcularse así:

Altura total del elemento = altura + relleno superior + relleno inferior + borde superior + borde inferior.

Si la anchura total de un elemento depende de los elementos que van en su interior, debemos sumar también los margenes al resultado total
