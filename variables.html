Variables CSS o Propiedades personalizadas CSS

Introducción

A menudo se las denomina "Variables CSS", pero ese no es su nombre real son propiedades personalizadas CSS "Custom Properties"

La variable son valores que se utilizan para no repetirse en este caso se podría considerar más una constante, pero temas de discusión aparte, se utiliza mucho para hacer un código css mucho más legible y comprensible.

Creación

Se inicia a través de un nombre antecedido de dos guiones dos puntos y un valor. Además, debe de estar siempre dentro de un selector. El selector elegido es de vital importancia pues determinara la especificidad o universalidad de su uso.

No debe contener caracteres especiales como &, @, ^, /

Por ejemplo:

--colorRojo: #FF0000;

Se pueden utilizar como valor de una variable otra variable ya declarada. Por ejemplo

html { --red: #a24e34; --green: #01f3e6; --yellow: #f0e765; --error: var(--red); --errorBorder: 1px dashed var(--red); --ok: var(--green); --warning: var(--yellow); }

Uso o referencia

Para hacer uso de una variable ya inicializada se debe usar la función var().

Por ejemplo:

:root{ --colorAzul:#0000AA; background-color:var(--colorAzul); }

Cabe destacar que si tuviéramos una variable inicializada en dos puntos diferentes y con valores diferentes el valor tomado sigue la filosofía de la cascada. Golpea o tiene efecto sobre el elemento el que más abajo o cerca se encuentra de este.

También podemos destacar que si la variable no ha sido inicializa o no se encuentra disponible podemos dar un valor en caso de ausencia. Por ejemplo:

:root{ --colorAmarillo:#FF00FF; } p{ color:var(--colorAmarillo, #AA00AA) }

Global versus local, Dinámico frente a estática.

En este apartado se pretende discutir el uso y la localización de la inicialización de las variables. Frecuentemente, se suele utilizar el selector (:root) que equivale al selector html, pero en caso de conflicto tendrá prioridad el primero sobre el último.

Ventajas de usar variables:

  • Código más comprensible y legible.
  • Los cambios en el código son mucho más rápidos.

Sobre escritura de variables

Una variable global que aparezca en el selector :root se puede sobreescribir en uno de sus hijos.

Uso de variables en media-queries.

Un ejemplo vale más que mil palabras:

:root { --blue: #1e90ff; --white: #ffffff; } .container { --fontsize: 25px; } /* Styles */ body { background-color: var(--blue); } h2 { border-bottom: 2px solid var(--blue); } .container { color: var(--blue); background-color: var(--white); padding: 15px; font-size: var(--fontsize); } @media screen and (min-width: 450px) { .container { --fontsize: 50px; } }
