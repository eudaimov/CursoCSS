Selectores

Los selectores son usados en css para seleccionar o encontrar un elemento o elemento al que queremos aplicar un determinado estilo.

Se pueden dividir en cinco categorías:

  • Selectores simples.
  • Selectores combinados.
  • Pseudo selectores de estado.
  • Pseudo selectores de partes.
  • Selectores de atributos.

Selectores simples

  • Selector de elemento simple

    • p {font-size:12pt;}
    h1 {font-size: 16pt;}
    h2 {font-size: 14pt;}

  • Selector de clase

    • .margin1em {margin-bottom:1em;}
    .negrita {font-weight: bold;}

  • Selector especifico de elemento+clase

    • p.textoAlineadoCentro{ text-align:center}; p.oculto {display:none;}

  • Selector de identificador único

    • div#principal {border: 2px solid black};

  • Selector de grupos

    • p, h1, h2{color: gray}

  • Selector universal

    • *{box-sizing: border-box}

Selectores combinados:

  • Selector descendente (espacio)

    • section p {margin: 1em;}

  • Selector hijos >

    • section > div.section2 {margin: 1em;}

  • Selector adyacente

    • div + p {background-image:yellow}

  • Selector hermanos adyacentes

    • p ~ ul{color:black}

    Nota importante Para poder el signo de equivalencia se tiene que mantener pulsada la tecla Alt mientras escribes 126

Pseudo selectores de estados:

Se utiliza para definir el estilo de un elemento mientras se encuentra en un estado en concreto

Por ejemplo, se puede utilizar para:

Aplicar estilo a un elemento cuando un usuario pasa el mouse sobre él

Estilo de enlaces visitados y no visitados de manera diferente

Dale estilo a un elemento cuando se enfoca

Para utilizar estas pseudo clases debe colocar dos puntos antes de definir el estado.

Pseudo selectores de partes ::

Un pseudo-elemento CSS se usa para diseñar partes específicas de un elemento.

Por ejemplo, se puede utilizar para:

  • Aplicar estilo a la primera letra, o línea, de un elemento.
  • Insertar contenido antes o después del contenido de un elemento.
Selector Ejemplo Descripción
::after p::after Insertar después de cada elemento <p>
::before p::before Insertar contenido antes de cada elemento <p>
::first-letter p::first-letter Selecciona la primera letra de cada elemento <p>
::first-line p::first-line Selecciona la primera línea de cada elemento<p>
::selection p::selection Selecciona la porción de texto párrafo que es seleccionado por un usuario
::marker li::marker Selecciona las viñetas, números o letras que componen una lista

Selector de atributos

Sirve para seleccionar elementos con atributos específicos.

Selector Ejemplo Descripción
[atributo] [target] Selecciona todos los elementos con el atributo target
[atributo=valor] [target=_blank] Selecciona todos los elementos cuyo atributo target sea igual a _blank
[atributo~=valor] [title~=flower] Selecciona todos los elementos cuyo atributo title contenga como valor la palabra "flower"
[atributo|=valor] [lang|=en] Selecciona todos los elementos cuyos atributo lang comience con el valor "en"
[atributo^=valor] a[href^="https"] Selecciona cada elemento<a> cuyo valor del atributo href comience con "https"
[atributo$=valor] a[href$=".pdf"] Selecciona cada elemento <a> cuyo valor del atributo href finalice con ".pdf"
[atributo*=valor] a[href*="w3schools"] Selecciona cada elemento <a> cuyo valor del atributo href contenga la palabra "w3schools"
