Z-Index

Introducción

La propiedad z-index se puede especificar con un valor entero (positivo, cero o negativo), que representa la posición del elemento a lo largo del eje z. Si no está familiarizado con el eje z, imagine la página como una pila de capas, cada una con un número. Las capas se representan en orden numérico, con los números más grandes sobre los más pequeños.

El nivel por defecto es 0

La utilidad de esta propiedad se observa en aquellos elementos cuya posición no es estática("static")

Sino estuviera descrita esta propiedad o dos elementos estuvieran en el mismo nivel y hubiera intersecciones entre los dos elementos, el último elemento en aparecer en la estructura HTML es el elemento que se visualizara en la parte superior.
