¿Dónde encontrarlo o situarlo?

Existe tres formas de aplicar nuestro cógido css.

  • CSS externo.
  • CSS interno.
  • CSS en linea.

Orden de prioridad

Si se han definido algunas propiedades repetidas para el mismo elemento en diferentes hojas de estilo, se utilizará el valor de la última hoja de estilo leída.

Ademas hay que tener en cuenta lo siguiente:

Todos los estilos en una página "caerán en cascada" en una nueva hoja de estilo "virtual" según las siguientes reglas, donde el número uno tiene la prioridad más alta:

  1. Estilo en línea (dentro de un elemento HTML)
  2. Hojas de estilo externas e internas (en la sección de cabecera)
  3. Predeterminado del navegador.

Por lo tanto, un estilo en línea tiene la prioridad más alta y anulará los estilos externos e internos y los valores predeterminados del navegador.

Dentro del mismo nivel también existe un orden de prioridad

  1. Elementos
  2. Clases
  3. Identificadores

CSS externo

El código css se escribe en un archivo externo a nuestro archivo html, la extensión de dicho archivo debe ser css y la vinculación se realiza dentro del elemento head mediante el elemento link

<link rel="stylesheet" href="css/style.css" type="text/css">

El atributo type especifica el tipo de medio del documento/recurso vinculado.

El valor más común de tipo es "text/css". Si omite el atributo type, el navegador buscará en el rel atributo para adivinar el tipo correcto. Entonces, si rel="hoja de estilo", el navegador asumirá que el tipo es "texto/css"

CSS interno

Se encuentra dentro del archivo html, pero dentro del elemento style y este a su vez dentro del elemento head

CSS en linea

Se puede encontrar a lo largo de todo en código html, formando parte del atributo style que puede ir en cualquiera de los elementos que nos puede aparecen en un body.

Su uso está desaconsejado y solamente recomendado para hacer pruebas de maquetación o evitar la aplicación de estilos globales que se usan desde otro lugar, aunque seguramente se puedan hacer en otro lugar que no sea este.
